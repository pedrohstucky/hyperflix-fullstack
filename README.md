# Hyperflix

Plataforma de streaming fullstack construída com **Java + Spring Boot** no backend e **TypeScript + React** no frontend.

## Sobre o projeto

O Hyperflix é uma aplicação web para catálogo e reprodução de conteúdo de vídeo, com foco em uma experiência moderna, escalável e segura.

## Tecnologias

### Backend

- Java
- Spring Boot
- JPA/Hibernate
- PostgreSQL

### Frontend

- React
- TypeScript
- Vite
- Tailwind/shadcn

## Funcionalidades esperadas

- Cadastro e login de usuários
- Catálogo de filmes e séries
- Busca por título/gênero
- Página de detalhes do conteúdo
- Reprodução de vídeo
